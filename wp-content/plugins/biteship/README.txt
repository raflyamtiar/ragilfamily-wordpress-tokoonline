Plugin Pengiriman dan Cek Ongkir untuk website WooCommerce yang dilengkapi dengan beragam pilihan ekspedisi untuk pengiriman Reguler, Instan, dan Kargo.

== Penjelasan Singkat ==

Plugin pengiriman yang membantu pebisnis online untuk mengirim barang setelah ada pembelian dari website WooCommerce.

Dilengkapi dengan berbagai pilihan ekspedisi yang memberikan kebebasan customer untuk memilih ekspedisi langganan mereka. 

<h3>Fitur</h3>

* Pengiriman yang didukung dengan 30 ekspedisi di Indonesia
* Kalkulasi total ongkos kirim di halaman check-out
* Bebas atur harga ongkos kirim (diskon atau mark-up)
* Fitur cek resi dengan domain milik Anda (https://yourdomain.com/tracking)
* Menambahkan lebih dari satu pick up point untuk membantu penjemputan di lokasi yang berbeda
* Bebas biaya API 
* Google Maps Pin Point (lokasi lebih akurat)
* Terintegrasi dengan dashboard web Biteship (monitor semua aktivitas pengiriman anda)

<h3> Kenapa Harus Biteship? </h3>

Plugin pengiriman Biteship menjamin semua pengiriman akan dikelola dengan baik, karena kami telah bekerja sama khusus dengan layanan ekspedisi untuk mengutamakan kepuasan pelanggan. Dengan semua fitur yang dibuat khusus berdasarkan riset, krik, dan saran dari pemilik bisnis online, kami yakin plugin ini sangat sesuai dengan kebutuhan semua pemilik bisnis online di Indonesia.

== Installation ==

Instalasi plugin Biteship mudah dan kurang dari 5 menit üòé. Berikut caranya:

1.Download dan activate plugin Biteship.
2.Isi data pemilik toko (nama toko, nama pengirim, nomor HP dan email pengirim).
3.Pilih apakah perlu aktivasi fitur Asuransi, COD atau set Berat Bawaan.
4.Isi alamat asal.
5.Registrasi dan buat kunci API di admin dashboard Biteship.
6.Pilih ekspedisi sesuai kebutuhan.
7.Centang aktivasi plugin dan klik ‚ÄúSave Changes‚Äù.
8.Buka halaman checkout toko kamu dan kamu bisa langsung cek ongkir sesuai ekspedisi.

Buat resi dan kirim barang juga mudah di Biteship üòé. Berikut caranya:

1.Top up saldo Bite Points sebagai deposito biaya kirim barang.
2.Buka halaman ‚ÄúOrders‚Äù.
3.Ubah order menjadi ‚ÄúProcessing‚Äù yang menandakan order customer sudah dibayar dan terverifikasi.
4.Klik ‚ÄúCreate Shipment‚Äù.
5. Resi otomatis muncul dan tunggu kurir jemput barang.

Jika kamu membutuhkan bantuan, bisa kontak ke info@biteship.com

== Frequently Asked Questions ==

= Bisakah saya menonaktifkan kurir yang tidak ingin saya pakai? =

Ya tentu, anda bisa mengaktifkan beberapa kurir yang sejalan dengan keinginan dan bisnis anda.

= Apakah ongkos kirim yang tercantum adalah ongkos kirim terbaru? =

Ya, karena Biteship terintegrasi langsung dengan perusahaan ekspedisi yang tercantum di kolom deskripsi. Jadi jika ada perubahan dari pihak ekspedisi, maka akan secara otomatis ter-update.

= Apakah plugin ini kompatibel untuk website saya? =

Ya, plugin Biteship kompatibel untuk penggunaan di Wordpress dan WooCommerce.

= Apakah plugin ini bisa melakukan pengiriman ke luar negeri? =

Sayang sekali saat ini plugin Biteship for WooCommerce hanya mendukung pengiriman di wilayah Indonesia saja.
